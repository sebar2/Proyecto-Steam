Introducción

Este menú mejorado ha sido diseñado específicamente para el proyecto Cerradura Smart RFID-BT.

¿Qué incluye?

  • Archivo CSS con estilos modernos (CSS/menu.css)
  • Archivo JavaScript con funcionalidades (Javascript/menu.js)
  • Componente HTML listo para usar (menu-component.html)
  • Instrucciones detalladas (INSTRUCCIONES_MENU.md)
  • Archivo de demostración (demo-menu.html)

Funcionalidades JavaScript

  • ✨ Efecto de scroll en el navbar
  • ✨ Cierre automático del menú móvil
  • ✨ Marcado de página activa
  • ✨ Smooth scroll para enlaces internos
  • ✨ Animaciones de dropdown
  • ✨ Navegación por teclado (ESC)
  • ✨ Resaltado de sección actual
  • ✨ Animación de entrada de items

Instalación

Nota: Los archivos ya están creados en tu proyecto. Solo necesitas agregarlos a tus páginas HTML.

Paso 1: Agregar CSS

En el <head> de tus páginas HTML:

<!-- Bootstrap CSS -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
<!-- Bootstrap Icons -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.0/font/bootstrap-icons.css" rel="stylesheet">

<!-- ⭐ Menú Mejorado ⭐ -->
<link rel="stylesheet" href="CSS/menu.css">

Paso 2: Agregar JavaScript

Antes de cerrar </body>:

<!-- Bootstrap JS -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>

<!-- ⭐ Menú Mejorado ⭐ -->
<script src="Javascript/menu.js"></script>

Paso 3: Reemplazar el Header

Copia el código del <header> desde menu-component.html y reemplázalo en:

  • ✅ index.html
  • ✅ avanze.html
  • ✅ inicio.html
¡Listo! Tu menú mejorado está funcionando.

Código para Personalizar

--primary-color: #2c3e50; /* Color principal */
--secondary-color: #3498db; /* Color secundario */
--accent-color: #e74c3c; /* Color de acento */
--text-light: #ecf0f1; /* Texto claro */
--hover-bg: #34495e; /* Fondo al hover */
}
