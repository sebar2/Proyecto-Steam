Componente de Menú

Este es el componente de menú mejorado para tu proyecto. Copia el código del <header> y pégalo en tus páginas HTML.

Características:

  • ✅ Diseño moderno y responsivo
  • ✅ Animaciones suaves
  • ✅ Menú desplegable para información
  • ✅ Indicador de página activa
  • ✅ Compatible con móviles
  • ✅ Accesible (navegación por teclado)
  • ✅ Smooth scroll para enlaces internos
Archivos Generados
1. CSS/menu.css

Archivo de estilos para el menú con animaciones y efectos modernos.

2. Javascript/menu.js

Script con funcionalidades avanzadas:

  • Efecto de scroll en el navbar
  • Cierre automático del menú móvil
  • Marcado de página activa
  • Smooth scroll para enlaces internos
  • Animaciones de dropdown
  • Navegación por teclado
  • Y más...
3. menu-component.html

Este archivo con el componente completo del menú listo para copiar.

Instrucciones de Uso
  1. Agregar los archivos CSS y JS a tus páginas: 
    <!-- En el <head> -->
<link rel="stylesheet" href="CSS/menu.css">

<!-- Antes de cerrar </body> -->
<script src="Javascript/menu.js"></script>
  2. Reemplazar el <header> existente:

    Copia el código del <header> de este archivo y reemplázalo en:

    • index.html
    • avanze.html
    • inicio.html
  3. Verificar que Bootstrap esté incluido:

    Asegúrate de que tus páginas tengan Bootstrap 5.3.3 y Bootstrap Icons.

  4. Probar el menú:

    Abre tus páginas en el navegador y verifica que el menú funcione correctamente.

Personalización

Puedes personalizar los colores editando las variables CSS en CSS/menu.css:

:root {
    --primary-color: #2c3e50;      /* Color principal */
    --secondary-color: #3498db;    /* Color secundario */
    --accent-color: #e74c3c;       /* Color de acento */
    --text-light: #ecf0f1;         /* Texto claro */
    --text-dark: #2c3e50;          /* Texto oscuro */
    --hover-bg: #34495e;           /* Fondo al pasar el mouse */
}
